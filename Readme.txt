How to run code:
0. Make sure you're running on Python 3.7 and not 3.8.
1. Install dependencies:
- numpy
- torch 
- torchviz
- scikit-learn
(or refer to the requirements.txt file)
2. Put the train.xml and test.xml file in the same directory as the proj_1.py file
3. Run the python code:
python proj_1.py
